# Google DDoS-Guard Parser

Парсит Google по запросу `"DDoS-Guard" -site:ddos-guard.net -site:ddos-guard.ru`
и собирает все результаты до конца пагинации.

## Структура папки

```
google_ddosguard/
├── parser.js               ← главный скрипт
├── package.json
├── README.md
├── scripts/
│   └── cookies/
│       └── google.json     ← сюда кладёте куки Google (опционально)
└── results/
    └── ddosguard_ДАТА.csv  ← сюда сохраняются результаты
```

## Установка

Если у вас ещё нет `node_modules` в корне проекта — установите зависимости:

```bash
# Из корня проекта (там уже есть puppeteer в node_modules)
cd ..
npm install

# Либо установить отдельно в папке парсера:
cd google_ddosguard
npm install
```

Если puppeteer уже установлен в родительской папке — достаточно запустить
скрипт напрямую, дополнительная установка не нужна.

## Запуск

```bash
node parser.js
# или
npm start
```

## Режимы работы

**1 — С куками (рекомендуется)**
- Загружает куки из `scripts/cookies/google.json`
- Меньше шансов получить капчу
- После прохождения капчи — автоматически пересохраняет куки

Как получить куки Google:
1. Откройте Chrome → F12 → Application → Cookies → google.com
2. Экспортируйте через расширение «EditThisCookie» или «Cookie-Editor» в JSON
3. Сохраните в `scripts/cookies/google.json`

**2 — Инкогнито**
- Без куков, каждый запуск как новый пользователь
- Выше вероятность капчи на большом числе страниц

## Обработка капчи

Если Google показывает капчу (найдено < 3 результатов):
1. Скрипт останавливается и ждёт
2. Вы вручную проходите капчу в открытом браузере
3. Нажимаете Enter в консоли
4. Скрипт повторяет парсинг текущей страницы и продолжает

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `P` | Пауза / Возобновление |
| `H` | Справка |
| `Ctrl+C` | Сохранить собранное и выйти |

## Результат

CSV-файл в папке `results/` с колонками:

| Страница | Позиция | URL | Заголовок | Описание |
|----------|---------|-----|-----------|----------|
| 1 | 1 | https://... | Название | Сниппет |

Файл автоматически называется `ddosguard_ГГГГ-ММ-ДД_ЧЧ-ММ-СС.csv`
и сохраняется с BOM (корректно открывается в Excel).
